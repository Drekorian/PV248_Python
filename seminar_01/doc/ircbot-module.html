Module ircbot

source code

This module is a mockup of a IRC bot. When executed, it runs an endless loop that can be interrupted with EOF keystroke or interrupt keystroke. Bot is capable of handling commands and filters.

Author: Marek Osvald

Version: 2012.1210

Since: 2012.0921
