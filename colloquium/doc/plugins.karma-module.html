f_karma(msg)

Checks the message for sequence 'something++' and 'something--', if found, it calls the change_karma function. Depending whether the command is valid, returns either done state or next state with appropriate message.

Parameters:
  • msg (str) - message to be filtered
Returns: state.next|state.done
