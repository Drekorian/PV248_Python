Module conf_parse

source code

This module parses command line options indicating server address, port and delay of an IRC Bot and also command line arguments specifying configuration files. The module is capable of merging these using priority command line > configuration files > defaults and properly handles multiple configuration files with different configuration sections.

When executed, prints all loaded configurations to the standard output.
